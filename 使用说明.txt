使用说明
============================================

  🌊 水境 (Water Realm)
  毛玻璃 + 水纹涟漪 + 彩虹人物屏蔽 互动体验


启动方式
--------

  macOS：   双击「启动水境.command」
  Windows： 双击「启动水境.bat」

  浏览器会自动打开，允许摄像头权限即可开始。


系统要求
--------

  • Python 3（必需，用于启动本地服务器）
    - macOS 自带 Python3
    - Windows 用户请访问 https://www.python.org/downloads/ 下载安装
      安装时务必勾选 ☑ "Add Python to PATH"

  • 现代浏览器（Chrome / Edge / Safari）
  • 摄像头（用于获取画面）
  • 网络连接（首次需加载人脸识别模型 ~6MB）


功能一览
--------

  主页面
    → 点击「身处水中」卡片进入体验

  水纹效果
    → 视频自动变为毛玻璃质感
    → 画面中物体移动时产生水纹滑动
    → 手指点按画面产生圆形涟漪扩散

  彩虹屏蔽 🆕
    → 右上角 👤 按钮上传主人物照片
    → 其他人物自动被彩虹覆盖
    → 非主人物走过之处残留彩虹轨迹
    → 垃圾桶按钮 🗑️ 清除全部残留

  底部液态玻璃栏
    → 📷 摄像头开关
    → 透明度滑块（0=不透明，100=全透明）
    → ⬜ 展开二级菜单（截屏 / 录屏）
    → 🗑️ 清除彩虹轨迹

  录屏
    → 输出 MP4 格式，可直接发朋友圈
    → 使用硬件 H.264 编码，画质优秀


快捷键
------

  ESC      返回主界面
  空格键   截屏


分享给朋友
----------

  将整个「水境」文件夹复制给朋友即可。
  朋友只需双击对应系统的启动文件，无需任何配置。
